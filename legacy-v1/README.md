# Legacy v1 external-WAV workaround

This directory preserves the original v1.0.0 installer, `PlaySoundW`
implementation, and vgmstream notices for historical reference.

It extracts the watch cue from the user's own `music.xwb` and is no longer
recommended. The repository root and latest release use the native XACT v2
implementation, which requires no extracted WAV files.
